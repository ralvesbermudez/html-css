TAGS

head = cabeçalho, configurações do documento HTML

title = vai dentro da "head" e será o título que aparece na aba do navegador

body = é o corpo da página, parte onde será disposta o conteúdo inserido no arquivo HTML

h1 = é a tag usada para inserir um título no corpo

hr = horizontal row, é a tag usada para inseir uma linha horizontal de ponta a ponta no corpo

p = é a tag usada para inserir parágrafos no corpo

br = break row, é tag usada para criar uma quebra de linha dentro de uma tag p




Obs.:

Para inserir os caracteres < e > em um texto sem que gere uma nova tag
use &+l+t+; para <
e
use &+g+t+; para >

Para inserir emojis basta usar &+#+x+códigodaemojipedia
